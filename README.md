# Sales-Data-Dashboard

An interactive Power BI business intelligence dashboard built for **AdventureWorks Bike Shop**, providing end-to-end visibility into sales performance, customer behavior, product metrics, and geographic distribution.

## 📊 Dashboard Pages

### 1. Executive Dashboard
High-level KPIs at a glance:
- **Revenue**: ₹24.9M | **Profit**: ₹10.5M | **Orders**: 25.2K | **Return Rate**: 2.2%
- Revenue trending line chart (Jan 2020 – Jan 2022)
- Orders breakdown by category: Accessories, Bikes, Clothing
- Top 10 products by orders, revenue, and return rate
- Monthly revenue, orders, and returns vs. previous month

### 2. Map – Sales by Region
- Bubble map showing order volume across **North America**, **Europe**, and **Pacific**
- Countries covered: United States, Canada, United Kingdom, France, Germany, Australia
- Region filter buttons for drill-down

### 3. Product Details
- Per-product deep dive with gauge charts for **Monthly Orders**, **Revenue**, and **Profit vs. Target**
- Price adjustment slider (what-if parameter)
- Profit trend: Total Profit vs. Adjusted Profit over time
- AI-powered report summary with trend insights and anomaly detection

### 4. Customer Details
- **17.4K Unique Customers** | ₹1,431 Revenue per Customer
- Revenue per customer trend over time
- Top 100 customers table (orders + revenue)
- Orders segmented by **Income Level** and **Occupation**
- Top customer spotlight card

## 🛠️ Tools & Technologies
- **Microsoft Power BI Desktop**
- **DAX** (Data Analysis Expressions) for calculated measures
- **Power Query** for data transformation
- Data Source: AdventureWorks sample dataset


## 🚀 Getting Started
1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. Clone this repository
3. Open `Sales_Dashboard.pbix` in Power BI Desktop
4. Explore the four report pages using the navigation panel

## 📌 Key Insights
- **Tires and Tubes** is the most ordered product type
- **Shorts** has the highest return rate among product types
- **Mr. Maurice Shan** is the top customer by revenue (₹12.4K, 6 orders)
- Revenue shows a strong upward trend from mid-2021 onward
